HTML5 — CSS3 & jQuery;

Deze site is een prima voorbeeld van mijn kennis op het gebied van web-design.

1

Responsive - brouwser based webdesign, van de grond af geschreven met alleen een text editor - vim - ;) garandeert maximale performance.

2

Responsive design: ipad, android of PC door het toepassen van responsive design maakt het niet uit op welke device deze site benaderd wordt.

3

jQuery is een snelle, kleine en feature-rijke JavaScript-bibliotheek. Het maakt dingen zoals HTML document navigatie en animatie met Ajax veel eenvoudiger. Het werkt met een veelheid van browsers.

PHP

PHP is een generieke scripting taal welke voornamelijk gebruikt wordt om dynamische html pagina of andere web content te generen op de server.

1

Super krachtig platform voor het generen van web content. Er zijn veel 'plugins' beschikbaar (bv PDF files genereren of grafieken) en anders maak je ze zelf met python.

2

Korte onwikkel tijd zorgt voor een snelle 'return of investment'.

3

Standaard ondersteuning van uit hosting providers. PHP worden vaak al standaard in een hosting pakket geleverd dus je betaalt er niets extra's voor.

powershell

Good ol'Powershell als je het nodig bent is hij er voor je. De oude makker waar je op terug kunt vallen als de rest het niet meer doet. Ik heb veel in powershell geschreven oa server monitoren met realtime display maar ook scomm plugins etc.

1

Makkelijk om te leren, beheren.

2

Op alle moderne windows bakken beschikbaar. Alleen op en linux omgeving wordt het een beetje lastig.

3

Er is heel veel script-baar in powershell of het nou op VM-Ware, ADS of MS-SQL gaat bijna alle miscrosoft producten hebben wel een powershell plugin. Onwijs schaalbaar, dus.

Python

Een favoriet in mijn tool-box is Python, flexibel makkelijk schaalbaar en voor bijna alle platformen beschikbaar. In het plaatje zie je een mini voorbeeld welke ik had gemaakt voor iemand die een 'ibood hunt' in de gaten wou houden. Makkelijk om te leren, beheren en vooral leuk om te doen.

2

3

Geen 'curly brackets'! ;) Verplicht formatting van de code zorgt voor oplossingen die door iedereen op dezelfde manier geschreven worden. Hierdoor zijn scripts zonder documentatie goed lees baar. Ook jaren later.

JAVA

Hier ligt mijn persoonlijke ambitie de komende jaren. Op korte termijn wil ik me hierin laten certificeren. Onwijs interessant en ik geloof dat Java een toekomst vast taal is door zijn brede draagvlak en diversiteit aan platformen. Links zie je een kleinstukje en aan anagram generator welke ik gemaakt had om wordfoud oplossingen te maken. Ik de toekomst zal ik de gehele source code van de voorbeelden beschikbaar stellen voor geïnteresseerden.

1

Java is ontworpen om makkelijk in het gebruik te zijn. Makkelijk om te schrijven compileren en te 'debuggen'.

2

Java is object georiënteerd. Maakt het mogelijk om modulaire en herbruikbare code te schrijven.

3

Platform onafhankelijk, letterlijk biljoenen devices gebruiken Java. Kijk maar naar Android devices, een en al Java.
